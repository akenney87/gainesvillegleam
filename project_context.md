# Gainesville Gleam - Project Context

## Business Overview
**Company Name:** Gainesville Gleam
**Industry:** Residential/Commercial Cleaning Services
**Location:** Gainesville, GA

## Project Type
Marketing website for a professional cleaning service business.

## Design Requirements

### Style: "Pristine Luxe"
- Sleek, airy aesthetic embodying the feeling of a freshly cleaned space
- Generous whitespace and refined spacing
- Premium, spa-like quality
- Modern with organic warmth

### Color Scheme
Natural, fresh palette reflecting cleanliness and calm:

| Color         | Hex       | Usage                          |
|---------------|-----------|--------------------------------|
| Sage          | `#7C9885` | Primary accent, CTAs           |
| Sage Light    | `#9DB4A4` | Secondary accent, gradients    |
| Sage Pale     | `#E8EFEA` | Hover states, focus rings      |
| Sage Wash     | `#F4F7F5` | Form backgrounds, tags         |
| Cream         | `#FDFCFA` | Section backgrounds            |
| Warm White    | `#FFFFFE` | Primary background             |
| Charcoal      | `#1A1F1C` | Primary text, dark sections    |
| Slate         | `#4A524C` | Body text                      |
| Mist          | `#8A938C` | Secondary text                 |
| Gold          | `#C4A962` | Featured testimonial stars     |

### Typography
- **Display Font:** Outfit (headings, buttons, labels)
- **Body Font:** DM Sans (paragraphs, form inputs)
- Clean, geometric sans-serif pairing
- High readability with distinctive character

## Website Sections
1. **Hero** - Bold headline with floating appointment card, animated sparkles
2. **Services** - 2x2 grid with numbered cards and SVG illustrations
3. **About** - Trust signals with vertical stat cards
4. **Testimonials** - 3-column layout with featured dark card
5. **Contact** - Two-column with info and form
6. **Footer** - Dark charcoal with navigation columns

## Tech Stack
- HTML5 (semantic markup)
- CSS3 (CSS Variables, Grid, Flexbox, backdrop-filter)
- JavaScript (Intersection Observer, form validation, parallax)
- Responsive design (mobile-first breakpoints)

## Key Features
- Floating pill-shaped navigation with blur
- Animated sparkle effects in hero
- Scroll-triggered reveal animations
- Custom form validation with notifications
- SVG iconography throughout
- Reduced motion support

## Target Audience
- Homeowners in Gainesville area
- Busy professionals
- Property managers
- Small business owners

## Brand Values
- Clarity & calm
- Attention to detail
- Eco-friendly approach
- Professional reliability
- Customer satisfaction
