# Gainesville Gleam - Project Context

## Business Overview
**Company Name:** Gainesville Gleam
**Industry:** Residential/Commercial Cleaning Services
**Location:** Gainesville, FL

## Project Type
Marketing website for a professional cleaning service business.

## Design Requirements

### Style
- Modern and professional
- Clean, minimalist aesthetic
- User-friendly and accessible

### Color Scheme
Colors reflecting cleanliness, freshness, and trustworthiness:

| Color         | Hex       | Usage                          |
|---------------|-----------|--------------------------------|
| Fresh White   | `#FFFFFF` | Primary background             |
| Clean Blue    | `#4A90D9` | Primary accent, CTAs           |
| Soft Teal     | `#5BBFBA` | Secondary accent               |
| Light Gray    | `#F5F7FA` | Section backgrounds            |
| Charcoal      | `#2D3748` | Primary text                   |
| Slate Gray    | `#718096` | Secondary text                 |

### Typography
- Clean, sans-serif fonts
- High readability
- Professional appearance

## Website Sections (Planned)
1. **Hero** - Main headline with call-to-action
2. **Services** - List of cleaning services offered
3. **About** - Company story and values
4. **Testimonials** - Customer reviews
5. **Contact** - Contact form and information
6. **Footer** - Links, social media, copyright

## Tech Stack
- HTML5
- CSS3 (modern features: Flexbox, Grid, CSS Variables)
- JavaScript (vanilla, for interactivity)
- Responsive design (mobile-first approach)

## Target Audience
- Homeowners in Gainesville area
- Busy professionals
- Property managers
- Small business owners

## Brand Values
- Reliability
- Attention to detail
- Eco-friendly options
- Professional service
- Customer satisfaction
